Greetings Galaxy Hitchhiker!

I hope you enjoy the reference to my favourite book. Anyway, great job.
This is the first programmer challenge. In your project there is a Flask BluePrint prepared which is called Holly. In Red Dwarf, Holly was an AI with 6,000 IQ. In our challenge, Holly won't be that smart. Your task is to implement three endpoints that will be tested before you can proceed:

  • /greetings - This endpoint is tested with a GET request and the expected response is string Hello, my friend
  • /sum - This endpoint is tested with a POST request. During the test, two values are passed to the form object (first_number and second_number). The task is to return their sum.
  • /redirect - This endpoint is tested with a GET request. In this case, you will need to implement the redirect to the /greetings endpoint in a proper Flask way (implementation of the /greetings endpoint is required for this step).
For more useful information, see the Flask documentation.
